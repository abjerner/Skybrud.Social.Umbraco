Instagram uses OAuth 2.0 for authentication and communication. In order for users to authenticate with the Instagram API, you must specify the ID, secret and redirect URI of your Instagram app. You can create a new app at the following URL: http://instagram.com/developer/clients/manage/

When a user authenticates with the Instagram API, an access token is saved along with information about the user. With the Instagram API, access tokens will not expire - unless the user deauthorizes the app.

Instagram apps will by default run in sandbox mode, which will restrict your access to the Instagram API. It is recommended to see the Instagram API documentation for further information about the restrictions caused by running in sandbox mode. If your Instagram app instead is running in live mode, the scopes listed below will require prior approval by Instagram. The public_content scope is required to request information or media from other users.
