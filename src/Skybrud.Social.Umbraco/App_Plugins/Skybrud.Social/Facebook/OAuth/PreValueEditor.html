Facebook uses OAuth 2.0 for authentication and communication. In order for users to authenticate with the Facebook API, you must specify the ID, secret and redirect URI of your Facebook app. You can create a new app at the following URL: https://developers.facebook.com/

Notice: When a user authenticates with the Facebook API, an access token is saved along with information about the user. With the Facebook API, access tokens will expire after roughly two months.

Notice: When a user is redirected to the Facebook login dialog, the scope (permissions of the app) is specified. It is currently not possible to specify a scope, but this will be possible in a future version of this package. The current scope sent to the login dialog is: read_stream user_status user_about_me user_photos.
