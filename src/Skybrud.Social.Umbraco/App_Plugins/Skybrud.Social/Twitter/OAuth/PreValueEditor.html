The Twitter API uses OAuth 1.0a for authentication and communication. In order for users to authenticate with the Twitter API, you must specify a comsumer key (Twitter calls it an API key) and a consumer secret (Twitter calls it an API secret) of your Twitter app. You can create a new app at the following URL: https://apps.twitter.com/

Notice: You may have to specify a callback URL for your Twitter app to enable OAuth authentication (the login dialog). The URL is not used by this package, so you can just specify a somewhat random URL.
